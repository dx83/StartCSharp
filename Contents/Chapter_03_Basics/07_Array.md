## 4. 배열 (Array)
```csharp
int[] product = new int[5];
string[] names = new string[1000];
```
- `[]` : 대괄호(square bracket), 해당 변수가 배열임을 의미
- `new` : 참조 형식과 사용되는 경우 그에 필요한 메모리를 힙(heap)에 할당하는 역할
