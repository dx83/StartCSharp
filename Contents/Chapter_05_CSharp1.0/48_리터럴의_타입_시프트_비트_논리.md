### 리터럴에도 적용되는 타입
> 코드 내에서 사용되는 리터럴도 그에 해당하는 타입이 적용

▼ 수 : int형의 인스턴스이고 값이 고정된 변수로 사용
````
Console.WriteLine(5.ToString() + 6.ToString());  // 출력 : 56
````
▼ 문자열 : string 타입의 인스턴스로 취급
````
Console.WriteLine("test".ToUpper());  // 출력 : TEST
````
